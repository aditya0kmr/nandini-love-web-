## Quick Start Setup

### Step 1: Clone & Setup
```bash
git clone https://github.com/aditya0kmr/nandini-love-web-.git
cd nandini-love-web-
npm install
```

### Step 2: Create Folder Structure
```bash
mkdir -p src/components/shared src/components/games
mkdir -p src/pages
mkdir -p src/context src/hooks src/styles src/utils src/data
mkdir -p .github/workflows
mkdir -p public/images public/sounds
```

### Step 3: Copy ALL files from the complete code below and create them

### Step 4: Run Locally
```bash
npm run dev
```

### Step 5: Deploy to GitHub Pages
```bash
npm run deploy
```
